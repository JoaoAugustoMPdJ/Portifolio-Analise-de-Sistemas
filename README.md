# Portfólio – Análise de Sistemas

Olá! Sou João Augusto, recém-formado em Análise e Desenvolvimento de Sistemas, 
com foco em análise de requisitos, modelagem de dados e SQL.

Este portfólio reúne projetos pessoais e estudos de caso, 
destacando o processo de análise de sistemas, estruturação de bancos de dados,
análise de dados com Python e desenvolvimento de dashboards para apoio à tomada de decisão.

## Áreas de Atuação
- Levantamento e documentação de requisitos
- Modelagem de dados (DER e modelo lógico)
- Normalização de banco de dados
- Consultas SQL e geração de relatórios
- Regras de negócio
- Análise de dados com Python
- Desenvolvimento de dashboards em Python (visualização e indicadores)
- Aplicações iniciais de Inteligência Artificial
